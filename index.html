IoT-based Ozone Layer Depletion Parameter Monitoring System

This project aims at monitoring the parameters that affect the ozone layer in a very cost-effective and efficient manner. This is done by sending a monitoring device designed to the ozone layer and getting it back safely using a parachute for reuse purposes.

Railway Track Crack Detection System.

This project aims at reducing the rail accidents which occur due to cracks in the railway tracks by automatically detecting the crack in the track via the ultrasonic sensor (for surface cracks) and density sensor(for inner cracks). Once a crack is detected by the Robot, an SMS along with the exact location is sent to the server.
